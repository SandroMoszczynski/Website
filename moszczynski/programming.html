This was all accesible through command line. One could also see a number of statistics such as bus route and graphical data on passenger density.
This was a personal project worked on for my DnD group. The campaign I was working on took place in a cold Tundra like environment above the arctic cirlce. Thus I created a simple python program that runs through command line and calculates weather, temparture, snowfall and gives the ammount of expected sunlight(if any) on a specified day.
My MSc project code. This code was taken from various papers and adapted to my needs. The project used a machine learning agent from the projective simulation branch of reinforcement learning to optimise surface codes of quantum memory. The agent was free to run a c program that simulated a quantum computers memory. This program would test a surface code and output the results into a txt file that my agent wait for and then read. It would then deliberate on the results and output a new command to the surface code program through command line. This would continue until the surface code passed a stability threshold that had been preset, or the number of allowed moves had been passed.
This was an asssed project from UCL. The tasks was to optimise a mathematical clustering problem from code given to us. We were then asked to run performance analysi between the optimised and non optimised code and out put performance data on it.
