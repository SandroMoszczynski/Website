My MSc project code. This code was taken from various papers and adapted to my needs. The project used a machine learning agent from the projective simulation branch of reinforcement learning to optimise surface codes of quantum memory. The agent was free to run a c program that simulated a quantum computers memory. This program would test a surface code and output the results into a txt file that my agent wait for and then read. It would then deliberate on the results and output a new command to the surface code program through command line. This would continue until the surface code passed a stability threshold that had been preset, or the number of allowed moves had been passed.

Description

This was an assesed problem from UCl. It focused on creating a simulation of many pedestrians in a bos. It used physical models of how pedestrians move in crowds to simulate an area around each person where they would not want to walk into each other. The model was also optimised using paralelisation through multiple cpus and gave a readout of compute times with various settings. The program could simulate two types of pedestrians, the first is moving continuously forward unless bumped into and the second was going to a set destination on the map. The model could support various sizes of bounding box with many thousands of pedestrians and a visualiser to show their locations (although good luck keeping track of one pedestrian with many thousand dots).
